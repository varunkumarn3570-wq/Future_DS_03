# Customer Subscription Behaviour Analysis Dashboard

## Overview
This project is an interactive Power BI dashboard built to analyze customer subscription behaviour and marketing campaign performance. The dashboard provides insights into customer conversion trends, subscription rates, campaign effectiveness, and customer segmentation.

The goal of this project is to help businesses understand customer behaviour, identify high-performing channels, and improve subscription conversion rates through data-driven insights.

## Objectives

- Analyze customer subscription behaviour
- Identify high-performing contact channels
- Detect customer drop-off patterns
- Understand conversion trends
- Discover top-performing customer segments
- Improve marketing campaign effectiveness

## Business Questions Answered

1. Where are customers dropping off in the subscription process?
2. Which contact channel generates the highest subscription rate?
3. Which customer segments contribute the most subscribers?
4. How can subscription conversion rates be improved?
5. Which areas require optimization for better campaign performance?

# Dashboard Features

## KPI Cards
- Total Customers
- Total Subscribers
- Total Non-Subscribers
- Previous Campaign Success Rate
- Overall Subscription Rate

## Visualizations

### Subscription Funnel / Channel Analysis
Analyzes subscription performance across different contact methods.

### Subscription Rate by Contact Type
Compares conversion rates between:
- Cellular
- Telephone
- Unknown contact types

### Subscription Distribution by Job Category
Identifies job categories with the highest number of subscribers.

### Monthly Subscription Trends
Tracks subscriber activity across different months.

### Contact Type and Credit Default Analysis
Analyzes subscription performance based on customer credit default status.

### Interactive Filters (Slicers)
Dashboard includes filters for:
- Contact Type
- Marital Status
- Job Category

---

# Key Insights

- Cellular contact channel has the highest subscription conversion rate.
- Management professionals contribute the highest number of subscribers.
- Unknown contact types show significantly lower conversion performance.
- Certain months show declining subscription activity.
- Improved customer follow-up strategies may increase conversion rates.
